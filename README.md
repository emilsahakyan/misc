## Install & configure nvim

### To install neovim https://github.com/neovim/neovim
```
  ./install.sh [options]
````
-h for availabel options
### To configure based on https://github.com/emilsahakyan/nvim

```
  ./config.sh [options]
````
-h for availabel options

Basic installation options
`./config.sh -f -c`
